PHP + MySQL + JQuery

This project was created from the ground up for the company that I worked for. It was built around the Wordpress template using custom designs and building custom pages to work with PHP Sessions and a separate MySQL database for members. Data was stored with 256 encryption for passwords and sensitive client data. Not all client data was held in MySQL, instead data was held in house and pushed to MySQL data tables with only partial data that was needed. Further important PDF document files were encrypted and provided for clients online. Other data API's were integrated into the website, such as Zoho V2 API for sales data, and Mapp Cloud API for marketing emails.

PHP Noteboard

Created as a subdomain for private member communication and notifications in a facebook/forum format. This subdomain site allowed member login using PHP and Sessions with MySQL database tables. Members were allowed to write posts, comments, and replies. Notifications were sent out with every new entry. Document files were allowed for members to post and download.

Zoho V2 API

Zoho is a site that maintains client data for sales teams around the globe to communicate and share data within company team members. To use this data, a custom PHP class was created to handle the Token Authorization of the API. Methods were further created for data updates, additions, and modifications from MySQL database tables to Zoho. This data was integrated into the Waveland website for search quieries and modifications.

Mapp Cloud API (Bluehornet API)

Similar to the Zoho API, a custom PHP class was created for handling Mapp Clould API data and integration into the Waveland company website. Mapp Cloud, which was formally called Bluehornet, handles client contact information for emai marketing purposes. Data from MySQL tables were scheduled to upload in bulk to Mapp Cloud API, and custom methods were created for querying, adding, and modifying data.
