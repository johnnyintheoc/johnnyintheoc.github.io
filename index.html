Node.js Website using Express Handlebars and MongoDB

This site was built with Node.js and uses Express for handling the routing. Express Handlebars framework was added as the view engine to show the use of templates through MVC model.

Data is stored on the cloud using MongoDB Atlas (www.mongodb.com). A Cluster with multiple Collection documents was set up using AWS.

Sample data was gathered using the Google APIs with the Youtube API library. Written with Node.js, Youtube channel data is queried and stored on the AWS Cluster using custom keywords ("puppies", "bouldering", "movie trailers").

It further uses Bootstrap 4 for a responsive design on mobile and desktop.

PHP Website with Sessions using MySQL database
Previous Next

Built for a former company in the Oil & Gas Energy sector to handle Client, Investor, Advisor, and Employee data with years in development.

Uses PHP and Sessions to handle member login and communications. MySQL to store data through multiple tables. Custom AES data encryption written for securing data, which was obtained and synched with in house database on MS VBA.

Built around the Wordpress framework and stripped down to create custom pages without addons for more security. Custom backend dashboard integrated for client and member management.

Search Engine Optimized with content and coding geared towards both long tail and short tail keywords. Further techniques used with custom htaccess modifications and dynamic page generations while keeping within limitations of SEC/FINRA rules and regulations.

Multiple tools and APIs integrated including Salesforce API, Zoho V2 API, and Mapp Cloud API (formally Bluehornet API).

Keywords: PHP, Javascript, JQuery, MySQL, Sessions, AES Data Encryption, APIs
