Projects

Noteboard
This noteboard was created on the subdomain for the company to communicate internally using a members area. Each member were allowed to leave a message on the noteboard allow all members to view and respond to. As messages were posted, an email system notified all members and were allowed to easily respond using the email link. Members could further upload documents, images, and change their profile photo.
Languages: PHP, Jquery, HTML/CSS
Database: MySQL
Security: SSL, internal company IP access only using htaccess file


Zoho V2 API Integration
This project involved writing a custom class to handle the token security that Zoho V2 required for web based interactions (search, insert, modify, etc). The token expired every hour, so database tables were required for handling and resuing tokens for optimal performance. Further, CSV files were created and exported to a secure location for integrating with the Bluehornet API (see below).
Language: PHP
Database: MySQL


Bluehornet API
This project involved writing a custom class for downloading data from Bluehornet as well as direct data updates using VBA over HTTPS.
Language: PHP, VBA
Database: MySQL
